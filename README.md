# Dungeons & Dice 
A dice-rolling app for use with dice-based RPG systems, which allows the user to create and save custom dice sets.

- Install `expo-cli`

```
sudo npm i -g expo-cli
```

- Install dependences

```
npm i
```

- Run `expo-cli`

```
npm start
```

- Download expo client on phone and scan QR code from dev sever
